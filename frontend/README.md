# Next.js + Ant Design + Tailwind CSS

Dự án Next.js đơn giản với Ant Design và Tailwind CSS.

## Công nghệ

- **Next.js 16** - React framework với App Router
- **Ant Design 6** - UI component library
- **Tailwind CSS 4** - Utility-first CSS framework
- **TypeScript** - Type safety

## Cài đặt

```bash
pnpm install
```

## Chạy dev server

```bash
pnpm dev
```

Mở [http://localhost:3000](http://localhost:3000) trong trình duyệt.

## Build production

```bash
pnpm build
pnpm start
```

## Ghi chú

- Flash CSS của Ant Design đã được xử lý bằng `@ant-design/nextjs-registry`
- Ant Design và Tailwind CSS có thể sử dụng cùng nhau mà không xung đột
